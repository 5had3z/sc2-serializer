PyTorch Example

class sc2_serializer.example.SC2Dataset(sampler: ReplaySampler, features: list[str] | None = None, timepoints: TimeRange = TimeRange(min=0, max=30, step=0.5))[source]

Example PyTorch Dataset for StarCraft II Data.

This sample dataloader reads a replay around the timepoint range specified and returns the requested [scalar, minimap] features and the game outcome from the player POV.

Parameters:

  • sampler (ReplaySampler) – Sampler that yields a replay file and index in that file.

  • features (list[str], optional) – List of step data features to use. (default: [minimaps, scalars])

  • timepoints (TimeRange) – Timepoints in the replay to sample from.
